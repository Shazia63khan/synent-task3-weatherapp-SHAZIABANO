# Weather App

A Python command-line application that fetches real-time weather information using the OpenWeatherMap API.

## Features

- Search weather by city name
- Displays temperature in Celsius
- Displays humidity percentage
- Shows weather description
- Real-time weather data using API

## Technologies Used

- Python
- Requests Library
- OpenWeatherMap API

## Requirements

Install the required library:

```bash
pip install requests
```

## Setup

1. Create an account on OpenWeatherMap.
2. Generate an API key.
3. Open `weather_app.py`.
4. Replace:
```text
    YOUR_API_KEY with your own key
```

## How to Run

```bash
python weather_app.py
```

## Example Output

```text
Enter city name: Delhi

Weather Information
------------------------------
City: Delhi
Temperature: 38.5 °C
Humidity: 42 %
Weather: haze
```
